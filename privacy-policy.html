⚔️ Politique de Confidentialité

Dernière mise à jour : 11 novembre 2025

En bref : Le Coeur du Dragon est un jeu qui respecte votre vie privée. Nous ne collectons aucune donnée personnelle, nous n'utilisons aucun tracker, et toutes vos données de jeu sont stockées localement sur votre appareil.

1. Introduction

Bienvenue dans "Le Coeur du Dragon", un jeu de rôle médiéval-fantastique inspiré par Legend of the Red Dragon. Cette politique de confidentialité explique comment nous traitons vos informations lorsque vous utilisez notre jeu.

2. Informations que nous NE collectons PAS

Le Coeur du Dragon est conçu pour respecter votre vie privée :

  • ❌ Nous ne collectons AUCUNE donnée personnelle
  • ❌ Nous ne demandons pas votre nom réel, email, ou numéro de téléphone
  • ❌ Nous n'utilisons AUCUN cookie de suivi ou analytics
  • ❌ Nous ne partageons AUCUNE information avec des tiers
  • ❌ Nous ne vendons AUCUNE donnée

3. Données stockées localement

Toutes vos données de jeu sont stockées localement sur votre appareil :

3.1 Stockage Local (LocalStorage)

  • Nom du personnage : Le nom que vous choisissez pour votre héros
  • Progression du jeu : Niveau, expérience, statistiques, équipement
  • Sauvegardes : Vos parties sauvegardées
  • Préférences : Paramètres audio et configuration multijoueur (optionnelle)

Important : Ces données restent sur votre appareil et ne sont jamais envoyées à nos serveurs ou à des tiers.

3.2 Service Worker et Cache

Le jeu utilise un Service Worker pour fonctionner hors ligne. Cela signifie que les fichiers du jeu sont mis en cache sur votre appareil pour permettre le jeu sans connexion Internet. Aucune donnée personnelle n'est impliquée dans ce processus.

4. Mode Multijoueur LAN (Optionnel)

Si vous choisissez d'utiliser le mode multijoueur sur réseau local (LAN) :

  • Les scores peuvent être envoyés à un serveur local sur votre réseau domestique
  • Seuls le nom de votre personnage et vos scores sont partagés
  • Les données restent sur votre réseau local et ne sont jamais transmises sur Internet
  • Cette fonctionnalité est complètement optionnelle et désactivée par défaut

5. Permissions

Le jeu ne demande AUCUNE permission spéciale :

  • ✅ Pas d'accès à votre caméra
  • ✅ Pas d'accès à votre microphone
  • ✅ Pas d'accès à vos contacts
  • ✅ Pas d'accès à votre localisation
  • ✅ Pas d'accès à vos fichiers personnels

6. Cookies et Technologies Similaires

Le jeu n'utilise AUCUN cookie de suivi ou de publicité. Le stockage local (LocalStorage) est utilisé uniquement pour sauvegarder votre progression de jeu sur votre appareil.

7. Services Tiers

Le Coeur du Dragon n'utilise AUCUN service tiers pour l'analyse, la publicité ou le suivi. Le jeu est complètement autonome et ne communique avec aucun serveur externe, sauf si vous activez manuellement le mode multijoueur LAN.

8. Sécurité des Données

Comme toutes vos données sont stockées localement sur votre appareil :

  • Vous avez le contrôle total de vos données
  • Vous pouvez les supprimer à tout moment via les paramètres du jeu ou de votre navigateur
  • Aucune transmission de données ne peut être interceptée car il n'y a pas de transmission

9. Droits des Utilisateurs

Vous avez le droit de :

  • Accéder à vos données : Vos sauvegardes sont exportables via le jeu
  • Supprimer vos données : Utilisez l'option "Nouvelle Partie" ou supprimez les données du navigateur
  • Transférer vos données : Exportez et importez vos sauvegardes sur différents appareils

10. Enfants

Le Coeur du Dragon est adapté à tous les âges. Comme nous ne collectons aucune donnée personnelle, il n'y a aucun risque pour la vie privée des enfants. Les parents peuvent laisser leurs enfants jouer en toute sécurité.

11. Hébergement

Le jeu est hébergé sur GitHub Pages (https://simondesjardinshogue.github.io/lecoeurdudragon/). GitHub peut collecter des informations techniques de base (adresse IP, type de navigateur) dans leurs journaux de serveur conformément à leur propre politique de confidentialité, mais ces informations ne sont pas accessibles par le développeur du jeu.

12. Code Open Source

Le code source du jeu est disponible publiquement sur GitHub (github.com/SimonDesjardinsHogue/lecoeurdudragon). Vous pouvez vérifier par vous-même que le jeu ne collecte aucune donnée et respecte votre vie privée.

13. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour. Comme nous ne collectons aucune donnée de contact, nous ne pouvons pas vous notifier directement des changements.

14. Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez :

  • Consulter le code source du projet sur GitHub
  • Ouvrir une issue sur le dépôt GitHub du projet
  • Contacter le développeur via GitHub : @SimonDesjardinsHogue

15. Juridiction

Cette politique de confidentialité est conforme au Règlement Général sur la Protection des Données (RGPD) de l'Union Européenne et aux lois canadiennes sur la protection de la vie privée.
